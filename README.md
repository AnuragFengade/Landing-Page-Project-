🚀 TechNova – Responsive Landing Page


📌 Overview

TechNova is a modern, fully responsive landing page built using HTML5 and Tailwind CSS. It represents a fictional AI-powered product designed to help developers and businesses automate workflows and scale efficiently.

This project focuses on clean UI design, responsiveness, and user experience.

🎯 Features

📱 Fully responsive design (mobile, tablet, desktop)

🎨 Modern UI with Tailwind CSS

🧭 Navigation bar with smooth layout

💡 Hero section with call-to-action buttons

⚙️ Features section with card layout

💰 Pricing section with plans

📣 Call-to-action (CTA) section

📌 Footer with basic details


🛠️ Tech Stack

HTML5 – Structure

Tailwind CSS (CDN) – Styling & responsiveness
